---
title: 'Fresnel Parameters'
---

Fresnel Parameters can be created easier with the static factory method:

```tsx
<standardMaterial
  specularPower={32}
  reflectionFresnelParameters={FresnelParameters.Parse({
    isEnabled: true,
    leftColor: [1, 1, 1],
    rightColor: [0, 0, 0],
    bias: 0.1,
    power: 1,
  })}
>
  <cubeTexture assignTo="reflectionTexture" rootUrl={'...'} />
</standardMaterial>
```
